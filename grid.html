Push the Grid

Pushing and pulling the grid allows you to place content of more important towards the top of the page "code wise" and move it for display purposes. Below I have specified which blocks are first based on its position in the source code, then you can see what this add-on is actually meant to achieve when applied to the display.



<div class="grid-3 push-3">
</div>

<div class="grid-3 pull-3">
</div>

Make a Pyramid

This is an example test case where you can center a portions of your grid for layout reasons. To do this simply add the class "centerGrid" to the element you want to center.



<div class="grid-3 centerGrid">
</div>
